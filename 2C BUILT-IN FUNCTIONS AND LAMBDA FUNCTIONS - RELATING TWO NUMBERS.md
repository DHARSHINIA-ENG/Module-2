## BUILT-IN FUNCTIONS

---

### AIM  
To Write a python program to find the 7 to power of 5 using built-in functions.

---

### ALGORITHM

1. Begin the program.  
2. Get the input x from the user
3. Get the input y from the user
4. use built-in function 'pow'(x,y)
5. Then print the value
---

### PROGRAM

```
x=7
y=5
z=pow(x,y)
print(z)

```
